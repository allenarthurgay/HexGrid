HexGrid
=======

Bad ass hex grid platform for building games


Features
========

 * Rendering in 3D with WebGL.
 * Picking!


Controls
========

:arrow_up: :arrow_down: :arrow_left: :arrow_right: --> Move camera

`A`/`Z` --> Zoom in/out

`S`/`X` --> Adjust the angle of the grid

`Mouse Click` --> Select a hex
